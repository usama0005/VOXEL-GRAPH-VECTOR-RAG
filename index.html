01 / Overview

What is this system?

VOXEL-GRAPH-VECTOR-RAG enables domain experts and non-technical stakeholders to interact with complex geotechnical digital twins using plain natural language — no Cypher queries, no SQL, no specialist tooling required. The system routes each query through a task classifier, executes a hybrid retrieval step over both a Neo4j knowledge graph and a FAISS dense vector index, then grounds an LLM response using task-specific anti-hallucination prompt templates. Results are visualized as interactive 3D voxel highlights inside Rhino.

03 / Features

Capabilities

🗣️
Natural Language Querying
Converse with geotechnical data using plain English. No Cypher, no SQL, no domain-specific query language.
🔀
Hybrid RAG Pipeline
Fuses Neo4j symbolic graph traversal with FAISS dense vector retrieval for maximal recall and precision.
🛡️
Anti-Hallucination Prompting
Task-specific prompt templates constrain LLM outputs to only retrieved evidence, preventing confabulation.
🧠
9-Way Task Classification
Automatic routing to specialized retrieval and generation strategies per query type.
🟥
3D Voxel Visualization
Query results are instantly highlighted in Rhino 3D, turning abstract database results into spatial insight.
⚙️
LLM-Agnostic Design
Modular helper architecture supports OpenAI GPT-4o, Anthropic Claude, or any compatible provider.

05 / Usage

Quick Start

bash · install
# Clone the repository
git clone https://github.com/YOUR_USERNAME/VOXEL-GRAPH-VECTOR-RAG.git
cd VOXEL-GRAPH-VECTOR-RAG

# Install dependencies
pip install neo4j==5.14.0 faiss-cpu==1.7.4 numpy pandas openai anthropic
python · example query
from neo4j_connector import Neo4jConnector
from openai_helper    import OpenAIHelper
from faiss_manager    import FAISSManager

# Initialise components
neo4j = Neo4jConnector(
    uri="bolt://localhost:7687",
    user="neo4j",
    password="your_password"
)
gpt = OpenAIHelper(api_key="your_openai_key")

# Run a filtering query
question = "Find voxels with high moisture content in layer M3"
voxels  = neo4j.get_high_moisture_voxels_with_count(threshold=40.0)
answer  = gpt.generate_answer(question, voxels['voxels'], 'FILTERING')

print(answer['answer'])
